WP Destinations gives you the ability to create a map on the front of the site with the ability to see the posts through using modals.

== Description ==
WP Destinations gives you the ability to add [geolocation](https://en.wikipedia.org/wiki/Geolocation) data to your blog posts.

This gives you the ability map where your posts are created from, or are in reference to, using the [Google Maps JavaScript API](https://developers.google.com/maps/).

NOTE: This plugin requires a [Google Maps JavaScript API key](https://developers.google.com/maps/documentation/javascript/get-api-key).

If you already have a Google Maps JavaScript API key, enter your API key in the plugin's options screen.

If you do not currently have a Google Maps JavaScript API key, you will need to obtain one from Google.

Google’s official instructions for creating Maps API keys can be found here:
https://developers.google.com/maps/documentation/javascript/get-api-key

== Installation ==
1. Upload the `wp-destinations` folder to your `/wp-content/plugins/` directory

2. Activate the \"WP Destinations\" plugin in your WordPress administration interface

3. Add your Google API Key to the plugin options
